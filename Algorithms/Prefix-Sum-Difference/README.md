# 前缀和与差分 | Prefix Sum & Difference

## 核心思想
前缀和与差分是一对“互逆”的操作，就像一个在时间轴上做积分，一个做微分。它们的本质都是用预处理来换取 $O(1)$ 的查询时间，把原本需要重复计算的东西，提前算好存下来。
- **前缀和**：把“区间查询”变成“两点做差”。预处理出前 $i$ 项的和，以后任何区间 $[l, r]$ 的和，都等于 $sum[r] - sum[l-1]$。
- **差分**：把“区间修改”变成“两个点的标记”。要对 $[l, r]$ 所有元素加 $s$，只需在差分数组上做 $diff[l] += s$ 和 $diff[r+1] -= s$，最后统一求前缀和还原。

## 学习心得
- **前缀和**：把累加结果提前存入数组，方便之后的查询，常用于求区间和，每次查询的复杂度均为O(1)
- **差分**：把相邻数之间的差值提前存入数组，方便之后的计算，当对某个区间的每个值都进行加/减时，只需要O(1)的修改差分数组两端点的值
- 容易出错的地方：差分数组下标越界——$r+1$ 可能超出数组范围，需要多开几位、注意初始化$sum[1]$和$diff[1]$
